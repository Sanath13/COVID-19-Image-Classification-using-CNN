Classification of Chest Radiographs

Description:

The Convolutional Neural Networks model for classifiation of chest radiographs has been deployed using Flask, a python framework for web application. The application takes a chest radiograph and classifies it into three classes namely, COVID-19, Normal/Healthy and Viral Pneumonia. The model returns the class name with higher predicted value among these classes. The proposed method has not undergone a clinical study. Thus it does not replace a medical diagnosis since a more thorough investigation could be done with a larger dataset. Under those circumstances, our work contributes to the possibility of an accurate, automatic, fast, and inexpensive method for assisting in the diagnosis of COVID-19 through chest X-ray images.
